Lam
===

**Lam** is a vim plugin for using [vim-fugitive][] quickly. It steals
[leader mappings][] from the Janus vim distribution, and adds a few
helpers for managing feature branches, viewing stashed changes, and
other bits and bobs.

These configurations are designed for my own use. Please use them for
all kinds of joy, but if it breaks the special joy of fixing it is left
to you. I don't expect to respond to maintenance requests.

[vim-fugitive]: https://github.com/tpope/vim-fugitive
[leader mappings]: https://github.com/carlhuda/janus#git-support-fugitive
